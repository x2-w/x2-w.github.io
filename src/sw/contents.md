### [mdbook](https://github.com/rust-lang/mdBook)

```bash
# 安装mdbook
cargo install mdbook --vers "^0.5.2" --root .

# 安装mdbook插件
cargo install mdbook-mermaid --root .
cargo install mdbook-admonish --root .
```
